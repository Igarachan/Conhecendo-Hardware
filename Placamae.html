Placa Mãe

O que é?

Uma placa-mãe, também conhecida como placa principal, é o componente principal de um computador que permite a conexão e a comunicação entre diferentes componentes de hardware. Ela é uma placa de circuito impresso que abriga o processador, a memória RAM, as placas de expansão, as interfaces de armazenamento e outros dispositivos periféricos.

Funcionalidades

Conexão do processador (CPU): A placa-mãe possui um soquete (socket) onde o processador é encaixado. Ele é responsável por executar as instruções do software e controlar as operações do sistema. Gerenciamento da memória RAM: A placa-mãe possui slots de memória onde os módulos de memória RAM são conectados. A memória RAM armazena temporariamente os dados que o processador precisa acessar rapidamente. Interface de armazenamento: A placa-mãe possui conectores para dispositivos de armazenamento, como discos rígidos (HDDs) e unidades de estado sólido (SSDs). Essas interfaces, como SATA e M.2, permitem a conexão e o acesso aos dispositivos de armazenamento para leitura e gravação de dados. Slots de expansão: A placa-mãe oferece slots de expansão, como PCIe e PCI, para permitir a instalação de placas adicionais, como placas de vídeo, placas de som, placas de rede e outras placas de expansão. Esses slots permitem a personalização e a melhoria das capacidades do computador. Interfaces de conectividade: A placa-mãe possui portas e conectores para permitir a conexão de periféricos, como teclado, mouse, monitor, alto-falantes, USB, Ethernet e outros dispositivos externos. Gerenciamento de energia: A placa-mãe fornece circuitos de gerenciamento de energia que regulam a alimentação elétrica para os componentes do sistema. Ela inclui o conector de alimentação principal, bem como conectores adicionais para alimentar componentes específicos.

Slots

Os principais slots de uma placa-mãe podem variar dependendo do formato e do chipset da placa, mas aqui estão alguns dos slots mais comuns encontrados em placas-mãe modernas:

  • Slot PCI Express (PCIe): É um slot de alta velocidade usado principalmente para conectar placas de expansão, como placas de vídeo, placas de som, placas de rede, entre outros. Existem diferentes versões do PCIe, como PCIe x16 (mais comum para placas de vídeo de alto desempenho), PCIe x1 e PCIe x4.

  • Slot de memória DDR: São os slots onde você conecta os módulos de memória RAM (Random Access Memory) do sistema. As placas-mãe modernas geralmente possuem vários slots para suportar configurações de memória de canal duplo ou quad-channel.

  • Slot M.2: É um slot compacto e de alta velocidade usado principalmente para instalar unidades de armazenamento SSD (Solid State Drive). Os SSDs M.2 oferecem desempenho rápido e são muito populares atualmente.

  • Slot SATA: São os slots usados para conectar unidades de armazenamento tradicionais, como discos rígidos (HDDs) e unidades de CD/DVD. Os cabos SATA são usados para fazer a conexão entre a placa-mãe e os dispositivos de armazenamento.

  • Slot PCI (Peripheral Component Interconnect): É um slot mais antigo, mas ainda presente em algumas placas-mãe. É usado para conectar placas de expansão mais antigas, como placas de som, placas de rede e placas de captura de vídeo.

  • Slot de CPU: Este slot é específico para encaixar o processador (CPU) na placa-mãe. Os sockets da CPU podem variar dependendo do tipo de processador utilizado (por exemplo, Intel LGA ou AMD AM4).
